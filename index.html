HTML5 Media Recorder API Demo

Record a 320x240 video

Works on:

  • Firefox 41
  • Chrome 47 (enable experimental Web Platform features at chrome://flags)

Issues:

  • Videos longer than several seconds won't (immediately) play on Firefox (Error shown: Video can't be played because the file is corrupt)
  • Chrome records no sound
