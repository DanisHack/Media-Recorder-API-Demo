HTML5 Media Recorder API Demo

Record a 320x240 video


Works on:

  • Firefox 41
  • Chrome 47,48 (enable experimental Web Platform features at chrome://flags)
  • Chrome 49

Issues:

  • Pause and Resume are not implemented by any browser at this time
  • Chrome implemented audio recording in Chrome 49

Containers & codecs:

  • The .webm container is used by both Firefox and Chrome
  • VP8 video on Firefox and Chrome 47,48 and 49
  • VP9 video on Chrome 49
  • Vorbis audio @ 44100 Hz on Firefox
  • Opus Audio @ 48000 Hz on Chrome 49
